# Project Guide

## Purpose

This workbook demonstrates a practical Excel workflow for inventory and sourcing analysis in a small hospitality setting.

## Suggested review order

1. Project Overview
2. Product Master and Supplier Master
3. Supplier Map
4. Inventory Snapshot and Inventory Valuation
5. Pivot Inventory By Category
6. Pivot Supplier Coverage
7. Audit Checks, Assumptions and Data Quality

## Main outputs

* Estimated inventory value by category
* Supplier coverage by product count
* Product-level stock and cost reference
* Documented assumptions and data-quality notes

## Important limitation

The workbook combines verified context with management estimates. Source Type and Confidence fields should be read before treating any value as verified.
